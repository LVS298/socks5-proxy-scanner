# SOCKS5代理扫描器 🔍

一个自动化扫描、验证和分类SOCKS5代理的工具，特别适用于内网源访问。

## 功能特性

- ✅ 多源扫描：支持免费代理源和360 Quake API
- ✅ 智能验证：SOCKS5连接性测试 + 内网源访问测试
- ✅ 自动分类：按有效性、省份、运营商、可访问源分类
- ✅ 多格式输出：JSON、TXT、CSV格式
- ✅ 高并发：多线程扫描，快速验证
- ✅ 安全可靠：API密钥通过环境变量管理
